Requirements
------------------

<ins>Need</ins>:
* PHP 7.2 or later
* `composer`
* `git`
* apache webserver

<ins>Optional</ins>:
* `nodejs` & `npm` for styling (`less`)
* `make` tool
* `docker` & `docker-compose` to run containers

Installation
-----------------

1. `git clone https://github.com/githabbe/ramverk1.git`
2. `composer install` inside project directory
